## 🚀 Déploiement sur GitHub Pages

### Étape 1 : Créer un repository GitHub
```bash
git init
git add .
git commit -m "Initial commit: MG Business e-commerce site"
git branch -M main
git remote add origin https://github.com/votre-username/mg-business-site.git
git push -u origin main
```

### Étape 2 : Activer GitHub Pages
1. Allez dans les **Settings** de votre repository
2. Descendez à **Pages**
3. Sélectionnez **main** comme branche source
4. Cliquez sur **Save**

Votre site sera accessible à : `https://votre-username.github.io/mg-business-site/`
